# 行为准则

English version: [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)

PaperLite 希望成为一个平静、实用、适合科研人和开发者参与的开源项目。参与者需要一起维护尊重和高效的讨论环境。

## 鼓励的行为

- 友善、耐心、表达清楚。
- 默认对方是善意的，同时也可以指出具体问题。
- 技术分歧围绕证据、取舍和用户影响展开。
- 尊重不同经验水平、语言能力和研究背景。
- 不要未经允许分享私人信息、凭证或未公开数据。

## 不接受的行为

- 骚扰、侮辱、威胁或人身攻击。
- 性化语言或不当关注。
- 基于身份、背景、所在地或能力的歧视和排斥。
- 未经允许公开私人联系方式、凭证、日志、数据集或截图。
- 在维护者要求回到主题后反复打断技术讨论。

## 执行方式

维护者可以编辑、隐藏、锁定或删除违反本准则的评论、issue、PR 或 discussion。对于让项目环境变得不安全或低效的参与者，维护者也可以临时或永久限制参与。

如果需要报告行为问题，请尽量使用私密 GitHub 渠道，例如仓库安全/advisory 联系方式，或可用的维护者私下联系方式。
