# Code of Conduct

Chinese version: [CODE_OF_CONDUCT.zh-CN.md](CODE_OF_CONDUCT.zh-CN.md)

PaperLite aims to be a calm, useful open-source project for researchers and builders. Everyone participating in this project is expected to help keep discussions respectful and productive.

## Expected Behavior

- Be kind, patient, and clear.
- Assume good intent while still naming concrete problems.
- Keep technical disagreements focused on evidence, tradeoffs, and user impact.
- Respect different levels of experience, language fluency, and research background.
- Avoid sharing private information, credentials, or unpublished data without permission.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Sexualized language or attention.
- Discrimination or exclusion based on identity, background, location, or ability.
- Publicly posting private contact details, credentials, logs, datasets, or screenshots without permission.
- Repeatedly derailing technical threads after maintainers ask to refocus.

## Enforcement

Maintainers may edit, hide, lock, or remove comments, issues, pull requests, or discussions that violate this code. Maintainers may also temporarily or permanently block participants whose behavior makes the project unsafe or unproductive.

If you need to report conduct concerns, use a private GitHub channel when possible, such as repository security/advisory contact paths or direct maintainer contact if available.
